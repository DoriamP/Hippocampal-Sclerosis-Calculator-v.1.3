## 🧠 Description

This calculator estimates the **probability of hippocampal sclerosis** from three morphometric measurements obtained by magnetic resonance imaging (MRI). The user enters the values ​​for the left and right hippocampus for each variable; The application automatically calculates the percentage difference between both hippocampi and applies the predictive model to estimate the probability.

### Input Variables

| Variable | Unit | Description |

|---|---|---|

| Hippocampal Volume (Left and Right) | mm³ | Volumetric volume of each hippocampus |

Dentate Gyrus Thickness (Left and Right) | mm | Granular Layer Thickness of the Dentate Gyrus |

| Hippocampal Head Coronal Angle (Left and Right) | degrees (°) | Coronal Tilt Angle of the Hippocampal Head |

The difference between sides is calculated as:

```
Difference (%) = |larger − smaller| / greater × 100
```

### Model Equation

```
logit(p) = −26.55 + 0.812 × DifVol + 1.128 × DifDenta + 0.230 × DifÁngulo

p = 1 / (1 + e^(−logit))
```

## 📊 Summary of Statistical Methodology

### 1. Study Design
- Observational case-control study.

- Total sample: 46 patients (21 with hippocampal sclerosis, 25 controls).

- Final analytical sample: **n = 40** (20 cases, 20 controls), after excluding 6 patients with incomplete data for angle/thickness variables.

### 2. Variable Selection
- **Univariate Analysis:** Mann-Whitney U test (non-parametric, suitable for small samples).

- **Selected Variables** (the three with the highest discrimination and complete data):

- Hippocampal volume difference (p < 0.0001)

- Dentate gyrus thickness difference (p < 0.0001)

- Hippocampal coronal angle difference (p < 0.0001)
- **Excluded Variables:** FLAIR intensity (35% missing data), CA1 thickness (high correlation with dentate gyrus), elongation (weak discrimination, marginal p = 0.027), gyrification (p = 1.0).

### 3. Model
- **Type:** Binary logistic regression with L2 regularization (ridge, C = 5).

- **Justification for regularization:** Near-perfect data separation in a small sample; The MLE coefficients do not converge without penalty (functional equivalent to Firth's method).

- **EPV (events per variable):** 6.7 (acceptable lower limit with regularization).

### 4. Internal Validation
| Metric | Value |

|---|---|

| Apparent AUC | 1.000 |

| AUC LOO-CV | 0.995 (95% CI: 0.977–1.000) |

| Bootstrap Optimism (2000 resamples) | 0.005 |

| Nagelkerke R² | 0.943 |

| Brier Score (LOO-CV) | 0.055 |

| Hosmer-Lemeshow p | 0.88 (good fit) |

### 5. Diagnostic Performance (LOO-CV)
| Metric | Value |
|---|---|

Sensitivity | 100% |

Specificity | 95% |

PPV | 95.2% |

NPV | 100% |

Accuracy | 97.5% |

Youden Index | 0.95 |

Optimal Threshold (Youden J) | p = 0.053 |

### 6. Software Used
- Python 3.11: `scikit-learn 1.x` (Logistic Regression), `statsmodels 0.14`, `scipy 1.x`

### 7. Limitations
- **Small sample size (n = 40):** Risk of overfitting despite favorable internal validation.

- **EPV = 6.7** is below the classic EPV threshold ≥ 10; regularization mitigates but does not eliminate the risk.
- **No external validation:** Results must be confirmed in independent cohorts.

- **Missing data:** Handled with full case analyses; possible selection bias.

## 🚀 Usage

### Option 1 — GitHub Pages (recommended)

1. Fork or clone this repository.

2. In **Settings → Pages**, select the `main` branch and the `/` root folder.

3. Access the URL generated by GitHub Pages (e.g., `https://username.github.io/hippocampal-sclerosis/`).

### Option 2 — Local

```bash
git clone https://github.com/user/hippocampal-sclerosis.git
cd hippocampal-sclerosis
# Open index.html in any modern browser
open index.html
```

No server or external dependencies required. Works offline.
